# 💻 Ejercicio Prueba Tecnica

Este proyecto es un ejercicio para una prueba técnica.

## 🛠 Scripts

```bash
# Instalar dependencias
$ npm install

# Encender servidor de desarrollo en localhost:3000
$ npm start

# Crear version de producción y lanzar el servidor
$ npm run build

```

## 💎 Tecnologías

- ReactJS
- Bootstrap
- Axios
